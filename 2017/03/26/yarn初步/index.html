0x02 从 npm 迁移到 Yarn

从 npm 迁移到 Yarn 很简单，只需要在原有的 npm 项目中运行 yarn init 命令就可以了，Yarn 会自动生成 yarn.lock锁文件，然后就可以使用上述 Yarn 命令来管理项目了。
